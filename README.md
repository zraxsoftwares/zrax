# ZRAX SOFTWARES

A high-performance brand landing page for **ZRAX SOFTWARES** — a Nairobi-based software engineering company. Built with Next.js 16, React 19, Framer Motion, and Tailwind CSS v4.

## Sections

- **Hero** — full-screen intro with animated backgrounds, tech badges, and CTAs
- **Services** — six service pillars (Web, Mobile, Cloud, AI, Security, UX)
- **Why Choose Us** — four value propositions with staggered animations
- **Portfolio** — three featured projects with tech stacks and hover effects
- **Ratings** — social proof with 4.9/5 star rating and testimonial
- **Contact** — inline contact form with validation + direct contact info

## Tech Stack

| Layer | Tech |
|-------|------|
| Framework | Next.js 16 (App Router) |
| UI Library | React 19 |
| Language | TypeScript 5 |
| Styling | Tailwind CSS v4 |
| Animation | Framer Motion 12 |
| Icons | Lucide React |

## Getting Started

```bash
# Install dependencies
npm install

# Copy environment variables
cp .env.example .env.local

# Start dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

See `.env.example` for all configurable values. All variables are prefixed with `NEXT_PUBLIC_` and are safe to commit as defaults. Override in `.env.local` for different environments.
